# VR Library Usage Guide

**Quick Start:**
1. Launch in WebXR-compatible browser
2. Click "Enter VR" button
3. Put on your VR headset

**Controls:**
- VR controllers: Grab and interact with objects
- Gaze: Select UI elements and buttons
- Movement: Teleport or smooth locomotion
